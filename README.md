Some personal scripts i use to edit images pair with a TKinter window

## Functions

- Multiplicate image into herself
- Invert image color
- Transparancy for white pixels
- Edge Ehance image
- Pixelate image
- Pixelate GIF image
- Extract images from GIF image
- Comic book effect
- Emboss effect
- Segmented effect
- Add watermark
